GYM COUNTER

Welcome to Gym Counter!


This system offers a simple yet effective solution for monitoring gym attendance using a compact Raspberry Pi-based setup. Designed to help visitors quickly assess how crowded the gym is, it uses physical IN/OUT buttons to count how many people are inside.

The system includes an auto-decrement feature — if someone forgets to press "OUT", the system assumes a standard workout time of 60 minutes and subtracts them automatically. This makes the counter more realistic and reliable, while keeping it easy to use.


Once installed, the system immediately starts monitoring gym activity through two physical buttons: one pressed at entry (IN), and one at exit (OUT). Each press updates the live counter.

The current number of people in the gym is shown via a simple web interface, accessible from any device on the same network. It updates in real time every few seconds, providing an always-accurate snapshot of gym occupancy.

The interface can also be extended to log timestamps and visualize trends, helping staff analyze peak hours and optimize usage of space and equipment.

Gym Counter is built using Raspberry Pi 5 and Python, with GPIO-connected momentary buttons and a lightweight Flask-powered web interface. The data is stored in a local SQLite database and can be visualized or extended with additional features.

This system is cost-effective, fully offline (no sensitive data transmitted), and highly adaptable — it can be used in gyms, libraries, labs, or any physical space where occupancy tracking is needed.

This solution bridges everyday gym needs with modern technology, serving as a scalable foundation for future smart facility management.
